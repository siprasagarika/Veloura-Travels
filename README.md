# 🌍 Veloura Travels Website

A modern and responsive travel agency website built using HTML, CSS, and JavaScript.

## ✨ Features

* Hero slideshow banner
* 12 destination cards
* Customer testimonials with ratings ⭐
* Destination highlights section
* Booking popup form
* Scroll animations
* Responsive design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
